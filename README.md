# Javaによるオセロ実装．

## オセロの遊び方
8×8の正方形の盤と、表裏を黒と白に塗り分けた平たい円盤状の石を使用する。それぞれ黒と白を担当する2人のプレイヤーが交互に盤面へ石を打っていき、最終的に盤上の石が多かったほうが勝ちとなる。相手の石を自分の石で挟んだときは、相手の石を裏返すことで、自分の石にする。
(Wikipedia参照)
